# Markdown Guide

## Headings
```markdown
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6
```
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

---

## Text Formatting
```markdown
*Italic* or _Italic_
**Bold** or __Bold__
***Bold and Italic*** or ___Bold and Italic___
~Strikethrough~
```
*Italic* or _Italic_
**Bold** or __Bold__
***Bold and Italic*** or ___Bold and Italic___
~Strikethrough~

---

## Blockquotes
```markdown
> This is a blockquote.
>> Nested blockquote.
```
> This is a blockquote.
>> Nested blockquote.

---

## Lists

### Unordered List
```markdown
- Item 1
  - Subitem 1.1
  - Subitem 1.2
- Item 2
```
- Item 1
  - Subitem 1.1
  - Subitem 1.2
- Item 2

### Ordered List
```markdown
1. First item
2. Second item
   1. Subitem 2.1
   2. Subitem 2.2
3. Third item
```
1. First item
2. Second item
   1. Subitem 2.1
   2. Subitem 2.2
3. Third item
